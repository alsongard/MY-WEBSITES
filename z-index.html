Complex z-index


To use z-index the following must be used:

  • The elements must have a position style that is fixed, absolute, relative and absolute and on flex child elements.
  • An element with a higher stack value will always be above or on top of an element with a lower stack value(index)

The code for the above is shown below: 

    <p>Hello World</p>
    <div style=" height: 200px; width: 1000px; position: relative; border:2px solid black;"> </div >
        < div style="background-color:  rgb(0, 140, 255); z-index: 5; top:2px; left:5px;"> </div>
        < div style="background-color: rgb(224, 19, 224);  z-index: 6; top:15px; left:20px;"> </div>
        < div style="background-color: green; z-index: 7; top:25px; left:35px;"> </div>
        < div style="background-color: yellow;  z-index: 8; top:35px; left:55px;"> </div>
        < div style="background-color: rgb(0,12, 211); z-index:9; top:45px; left:75px;"> </div>
        < div style="background-color: rgb(200,15,15); z-index:10; top:55px; left:95px;"> </div>
